# Product Catalog

This applications shows a list of products and their details. 

## Table of Contents
- [Features](#features)
- [Technologies Used](#technologies-used)
- [Visual Appearance](#visual-appearance)
- [Setup Instructions](#setup-instructions)


## Features
- Shows a list of products using mock API data
- It has Product details modal
- Search products
- Implented debounce for search to avoid multiple API calls
- Implemented custom hooks for fetching data
- Responsive design
- Used Tailwind CSS for styling
- Used React Rating Stars Component for rating
- Used Next.js for server side rendering


## Technologies Used
- Next.js
- Tailwind CSS
- React

## Setup Instructions
1. Clone the repository using `git clone` or download zip file and navigate to the project directory.
2. Install dependencies using `npm install`.
3. Start the development server using `npm run dev`
4. Open `http://localhost:3000` in your browser
